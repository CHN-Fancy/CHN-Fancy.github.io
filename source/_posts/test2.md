# 安装Linux

## 安装所需工具

测试图片：

![3wvBng.jpg](https://s2.ax1x.com/2020/02/27/3wvBng.jpg)

生成静态文件

```
hexo generate
```

启动服务器

```
hexo s
```

部署deploy

```
hexo deploy

**也可以简写**
hexo d 
```

| 参数             | 描述                     |
| ---------------- | ------------------------ |
| -g`, `--generate | 部署之前预先生成静态文件 |
